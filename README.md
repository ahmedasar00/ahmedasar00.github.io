# ahmedasar00.github.io

Personal portfolio website for **Ahmed Mohamed Asar** — Software Engineer.

🌐 Live site: [https://ahmedasar00.github.io](https://ahmedasar00.github.io)

## Stack

- Pure HTML5 + CSS3 + Vanilla JavaScript (no frameworks)
- Hosted on GitHub Pages

## Theme

- Black space background with twinkling stars (✨)
- Classic silver/white accents
- Split-view layout (two columns side by side)
- Fully responsive (mobile + desktop)

## Files

- `index.html` — page structure & content
- `styles.css` — theme, layout, animations
- `script.js` — generates stars + small UX touches
